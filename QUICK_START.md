# 🚀 Quick Start Guide

## ⚡ **Instant Launch (Recommended)**
```bash
Double-click: start-app.bat
```

## 💻 **Manual Start (Alternative)**
```bash
# Terminal 1: Start React Server
npx vite --host

# Terminal 2: Start Desktop App (after server starts)
npx electron .
```

## ✅ **Success Indicators**
- ✅ Vite server shows: `Local: http://localhost:5173/`
- ✅ Electron window opens with blue sidebar
- ✅ Dashboard displays statistics cards

## 🛑 **To Stop the Application**
- Close the Electron window
- Press `Ctrl+C` in the terminal

## 📱 **First Time Setup**
1. Install Node.js from [nodejs.org](https://nodejs.org)
2. Run: `npm install`
3. Double-click: `start-app.bat`

**You're ready to go!** 🎓📅
